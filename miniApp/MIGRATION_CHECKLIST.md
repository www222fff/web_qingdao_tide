# Taro小程序转换检查清单

在发布小程序前，请完成以下检查项目：

## 项目配置

- [ ] 修改 `project.config.json` 中的 `appid`
- [ ] 验证 `miniprogramRoot` 指向正确的构建输出目录
- [ ] 检查小程序名称和描述

## 网络配置

- [ ] 在微信小程序后台添加请求域名白名单：
  - [ ] `https://marine-api.open-meteo.com` (潮汐数据API)
  - [ ] 其他第三方API域名（如有）

- [ ] 测试网络请求是否正常

## 构建和部分测试

- [ ] 运行 `npm run build:weapp` 检查编译是否成功
- [ ] 检查 `dist/weapp/` 目录是否生成
- [ ] 使用微信开发者工具打开 `dist/weapp/` 目录
- [ ] 验证页面是否正确加载

## 功能测试

- [ ] 测试潮汐数据是否正确加载
- [ ] 验证图表是否正确显示
- [ ] 检查高潮/低潮标记是否准确
- [ ] 验证农历日期转换是否正确
- [ ] 测试摄像头图像是否显示（需要图片URL可访问）

## 页面测试

- [ ] 首页加载
- [ ] 数据加载中状态
- [ ] 数据加载失败处理
- [ ] 下拉刷新功能（如已实现）
- [ ] 页面滚动流畅性

## 性能优化

- [ ] 检查小程序包大小
- [ ] 验证首屏加载时间
- [ ] 测试内存使用情况
- [ ] 检查是否有内存泄漏

## 兼容性测试

- [ ] 测试iOS微信小程序
- [ ] 测试Android微信小程序
- [ ] 测试不同屏幕尺寸的适配
- [ ] 测试网络弱网环境下的表现

## 代码质量

- [ ] 代码审查
- [ ] 检查TypeScript类型检查是否通过
- [ ] 验证没有console.log调试语句（生产环境）
- [ ] 检查错误处理是否完善

## 资源管理

- [ ] 压缩和优化图片资源
- [ ] 移除未使用的依赖
- [ ] 检查图片路径是否正确
- [ ] 验证所有静态资源都被正确打包

## 安全性

- [ ] 验证没有硬编码的密钥或敏感信息
- [ ] 检查API请求是否使用HTTPS
- [ ] 验证用户隐私政策
- [ ] 检查是否需要用户授权（如位置信息等）

## 文档更新

- [ ] 更新README.md中的说明
- [ ] 更新版本号
- [ ] 记录版本更新日志
- [ ] 添加使用说明文档

## 发布准备

- [ ] 准备小程序图标
- [ ] 编写应用介绍和描述
- [ ] 获取必要的资质证件
- [ ] 提交微信小程序审核

## 发布后

- [ ] 监控用户反馈
- [ ] 检查错误日志（Sentry等）
- [ ] 验证数据统计正确
- [ ] 准备热更新计划

## 可选优化

- [ ] 实现数据缓存和离线支持
- [ ] 添加用户反馈功能
- [ ] 实现分享功能
- [ ] 添加开屏广告（如需货币化）
- [ ] 实现用户登录和数据同步
- [ ] 添加更多地区的潮汐数据支持
